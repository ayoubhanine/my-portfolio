## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/ayoubhanine/my-portfolio.git
cd my-portfolio
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run frontend

```bash
npm run dev
```

---

### 4. Run backend

```bash
cd backend
node server.js ou npm start
```

---

## 🔐 Environment Variables
